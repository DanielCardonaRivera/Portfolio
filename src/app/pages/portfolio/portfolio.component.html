Sitio web de fundación Eliseo

Es el sitio web de una fundación de ayuda psicológica, la cual contiene cuatro secciones: Inicio, Quienes somos, Blog y Donaciones. En él se ofrece información de interés mediante carruseles acumulados y se crea el registro de donantes para la fundación. Las tecnologías utilizadas fueron React y Bootstrap.
